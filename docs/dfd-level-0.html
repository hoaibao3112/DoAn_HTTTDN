📝 Giải thích sơ đồ DFD Mức 0:

  • Hệ thống trung tâm (Process 0): Hệ thống quản lý nhà sách tổng thể
  • 8 External Entities (Thực thể bên ngoài):
    • Admin/Quản lý: Quản trị toàn bộ hệ thống
    • Nhân viên Kho: Quản lý sản phẩm, nhập hàng, kiểm kê
    • Nhân viên Bán hàng: Tạo hóa đơn, bán hàng POS
    • Nhân viên Thường: Chấm công, xin nghỉ, xem lương
    • Khách hàng: Mua hàng, tích điểm, trả hàng
    • Nhà cung cấp: Cung cấp hàng hóa, nhận thanh toán
    • Cổng thanh toán: Xử lý thanh toán online
    • Báo cáo/Thống kê: Module xuất báo cáo
  • Data Flows (Luồng dữ liệu): Các mũi tên thể hiện luồng thông tin giữa các thực thể và hệ thống
